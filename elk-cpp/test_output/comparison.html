Root Cause

Topological Sort Order:

  • C++: DFS produces: s1, p3, p2, t1, s0, p1, p0, t0
  • JavaScript: Maintains closer to input order: s0, s1, p0, p1, p2, p3, t0, t1

Why Different?

Topological sort is not unique - multiple valid orderings exist for any DAG. The C++ DFS-based approach and JavaScript's approach produce different but equally correct orderings.
